## Landing Page → Lead Generation Platform

| Field     | Definition                                                  |
| --------- | ----------------------------------------------------------- |
| Audience  | Service businesses needing qualified leads (dental, construction, real estate) |
| Problem   | Wasting ad spend on unqualified visitors, no follow-up automation |
| Outcome   | Turn website visitors into booked appointments automatically |
| Proof     | Vertical-specific landing pages, lead capture forms, SEO-optimized content |

### Key messaging
- "Convert more visitors into booked clients"
- "Industry-specific landing pages that rank"
- "Built for plumbers, dentists, and contractors"

## Positioning Rules

1. **Never lead with technology** — "Next.js" and "Prisma" are footnotes, not headlines
2. **Always lead with outcome** — the first sentence must name a business result
3. **One audience per demo** — every demo targets one primary buyer persona
4. **Proof is behavioral, not technical** — show workflows (catalog, checkout, audit), not architecture diagrams
5. **Avoid jargon that requires explanation** — "state machine governance" is internal language; "tracked at every state" is client language
