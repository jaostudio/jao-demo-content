# Demo Taxonomy

Each demo maps to one or more production system archetypes:

| Demo | Primary Archetype | Secondary Archetypes |
|---|---|---|
| landingpage-demo | Marketing / Acquisition (1.1-1.4) | Vertical engines, SEO landing systems |
| web-application-demo | SaaS Workflow (2.2) | Admin panels, operational tools, real-time collab |
| database-security-demo | Security / Identity (9.1-9.4) | Subscription SaaS, compliance, secure portals |
| commerce-demo | Transactional Commerce (3.1-3.4) | Trust systems, digital goods |
| marketplace-demo | Network Effects (4.1-4.4) | Booking, listings, reputation |
| content-platform-demo | Content / Publishing (5.1-5.4) | SEO engine, AI content |

## Archetype Reference

1. **Marketing / Acquisition** — traffic-to-conversion machines
2. **SaaS / Multi-tenant** — logged-in product experiences
3. **Transactional Commerce** — real money movement
4. **Marketplace / Network Effects** — two-sided supply + demand
5. **Content / Publishing** — editorial + SEO architecture
6. **Real-time Systems** — WebSocket-driven state sync
7. **Data / Analytics** — engineering-heavy credibility builders
8. **Developer / Platform** — API management, webhooks
9. **Security / Identity** — auth, RBAC, compliance
10. **Operational / Internal** — admin tools, back-office
11. **AI-Powered** — assistants, content generation
12. **Infrastructure UI** — job queues, system health
13. **Niche / Industry-specific** — healthcare, legal, education
